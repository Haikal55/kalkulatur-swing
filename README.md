# Kalkulator Pintar (Java Swing)

Aplikasi **kalkulator desktop berbasis Java Swing** yang dibuat menggunakan **NetBeans GUI Builder (JFrame Form)**.  
Aplikasi ini mendukung perhitungan ekspresi matematika beruntun dengan tampilan modern (FlatLaf) dan mode gelap.

---

## Fitur Utama
- Operasi dasar: `+  -  *  /`
- Dukungan ekspresi berantai  
  Contoh: `7+7-3+8*2`
- Angka negatif dan minus ganda  
  Contoh: `-1--2`
- Persen (`%`) sesuai perilaku kalkulator standar  
  Contoh:  
  - `10%` → `0.1`  
  - `100-20%` → `99.8`
- Tombol `CE`, `Backspace`, dan `+/-`
- Tampilan **Dark Mode** menggunakan FlatLaf

---

## Teknologi
- **Java SE 8**
- **Java Swing**
- **NetBeans GUI Builder**
- **FlatLaf (Dark Theme)**
